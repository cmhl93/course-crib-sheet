##  HTML Boilerplate and Comments

  * Every HTML Document we create will start with this boilerplate:
    ```
    <!DOCTYPE html>
    <html>
    <head>
    <!--Our metadeata goes here-->
      <title></title>
    </head>
    <body>
    
    <!--Our content goes here-->
    </body>
    </html>
    ```

  * Comments
    ```
    <!--Some content-->
    ```
  * <title> - name of tab, search engines
